# Padding Widget

A widget which pads its child.

```json
{
    "type": "padding",
    "padding": 16.0,
    "padding_vertical": 8.0,
    "padding_horizontal": 4.0,
    "padding_left": 4.0,
    "padding_right": 4.0,
    "padding_top": 4.0,
    "padding_bottom": 4.0
}
```

You must specify either:

* `padding`
* `padding_vertical` and `padding_horizontal`
* `padding_left`, `padding_right`, `padding_top`, and `padding_bottom`
