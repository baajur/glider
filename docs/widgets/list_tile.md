# List Tile Widget

A Material Design list tile.

```json
{
    "type": "list_tile",
    "title": "John Smith",
    "subtitle": "john@smith.com",
    "leading": {},
    "trailing": {},
    "destination": {}
}
```

The `destination` widget must be a [`scaffold`](scaffold.md).
