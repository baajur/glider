# Installation

Glider is not currently available via `pub`, but you can add it as a `git` dependency like this:

```yaml
glider:
  git:
    url: git://github.com/netsells/glider.git
```

There is more information about Git dependencies [here](https://www.dartlang.org/tools/pub/dependencies#git-packages).
